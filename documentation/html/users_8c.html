float calculateAverageRating ( Empresa  empresa)

Calculates the average rating for a specific company.

This function calculates the average rating based on the ratings stored in the 'Classificacao' structure associated with a specific company. If the company has received ratings, the function calculates and returns the average rating; otherwise, it returns 0.0.

Parameters
empresaThe 'Empresa' structure representing the company.
Returns
The average rating for the company. If no ratings are available, returns 0.0.
