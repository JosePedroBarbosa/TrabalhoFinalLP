Detailed Description

A header file providing utility functions for user input validation and retrieval.

Author
ESTG
Date
28-10-2023
Version
1
Copyright

This header file contains a collection of functions designed to facilitate user input in a C or C++ program. It includes functions for obtaining integers, floats, doubles, characters, and strings from the user while performing input validation to ensure that the entered values are within specified ranges or constraints.
